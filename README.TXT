Nombre del Proyecto: Teacher Planner

Descripcion del Proyecto: Este es un proyecto de Planificacion de Docente desarrollado en Python Flask. Proporciona varias planificaciones para los Docente con reporte.

Paso de Ejecucion del Proyecto:

1. Instalacion de Dependencias: Asegurate de tener instalado Python en tu sistema. Luego puedes instalar las dependencias del proyecto utilizando pip:
    -pip install -r requeriments.txt

2. Configuracion de la Base de Datos: Asegurate de tener un servidor de Base de Datos MySQL disponible. En el archivo app.py, se utiliza la libreria flask_mysqldb para la conexion  a la Base de Datos MySQL.

3. Ejecucion del Servidor: Una vez configurada la Base de Datos, puedes ejecutar el servidor Flask con el siguiente comando:
    -python app.py

4. Acceso a la Aplicacion: Abre tu navegador web y visita la direccion http://localhost:5000 para acceder a la aplicacion de Planificacion de Docentes.


Librerias Utilizadas:

    1. flask: Framework web para Python.

    2. flask_wtf: Extensión de Flask para integrar WTForms, una librería de formularios en Flask.

    3. wtforms: Librería para la creación y validación de formularios en Flask.

    4. flask_login: Manejo de sesiones de usuario y autenticación en Flask.

    5. flask_mysqldb: Extensión de Flask para integrar MySQL en la aplicación.

    6. flask_mail: Extensión de Flask para enviar correos electrónicos.

    7. os: Librería estándar de Python para interactuar con el sistema operativo.

    8. pdfkit: Librería para generar archivos PDF a partir de HTML.

    9. smtplib: Módulo de la librería estándar de Python para enviar correos electrónicos utilizando el protocolo SMTP.

    10. email.message: Módulo de la librería estándar de Python para manipular mensajes de correo electrónico.

    11. wtforms.validators: Módulo de WTForms que contiene validadores predefinidos para campos de formularios.
